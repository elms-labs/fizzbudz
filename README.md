## IMP!: Both client and sever has node_modules and package.json

### If you wanna work on client 
- <code>cd client</code>
- <code>npm install</code> 
- <code>cd client && npm run dev</code>

---

### If you wanna work on server 
- <code>cd server</code>
- <code>npm install</code> 
- <code>cd server && npm run dev</code>
